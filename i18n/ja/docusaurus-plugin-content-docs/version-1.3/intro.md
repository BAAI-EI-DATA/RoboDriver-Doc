# チュートリアルの概要

**5分以内**で、**Docusaurus** を素早く体験しましょう。

## クイックスタート

**新しいサイトを作成**して始めてください。

あるいは、すぐに **[docusaurus.new](https://docusaurus.new)** から Docusaurus を試すこともできます。

### 前提条件

- [Node.js](https://nodejs.org/en/download/) バージョン 20.0 以上：
  - Node.js のインストール時には、依存関係に関連するすべてのオプションをオンにすることを推奨します。

## 新しいサイトの生成

**クラシックテンプレート**（classic template）を使用して、新しい Docusaurus サイトを生成します。

次のコマンドを実行すると、クラシックテンプレートがプロジェクトに自動的に追加されます：

```bash
npm init docusaurus@latest my-website classic
```

このコマンドは、コマンドプロンプト、PowerShell、ターミナル、あるいはコードエディタに統合されたターミナルなど、任意の環境で入力できます。

また、このコマンドにより、Docusaurus を実行するために必要なすべての依存パッケージが自動的にインストールされます。

## サイトの起動

開発用サーバーを起動します：

```bash
cd my-website
npm run start
```

`cd` コマンドは、現在の作業ディレクトリを変更するために使います。作成した Docusaurus サイトを操作するには、まずターミナル上でそのディレクトリに移動する必要があります。

`npm run start` コマンドにより、ローカルでウェブサイトがビルドされ、開発用サーバーが起動します。その後、ブラウザで http://localhost:3000/ にアクセスすると、サイトを確認できます。

`docs/intro.md`（このページ）を開いて数行編集してみてください。サイトは**自動的に再読み込み**され、変更内容が即座に反映されます。
